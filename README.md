# BunnyEra AI — Modular Intelligence Engine
### BunnyEra LLC · 2025

BunnyEra AI 是 BunnyEra 品牌体系中的独立智能模块，负责多智能体协作、自动化逻辑、任务规划、代码生成、审查与执行。
本仓库采用企业级模块化结构，并与 BunnyEra Console、System、Matrix 等核心系统深度联动。

---

## Overview

This repository contains the AI-driven core of the BunnyEra ecosystem, including:

- Multi-agent architecture
- Prompt engineering system
- Automation logic
- Task planning and execution
- Integration with BunnyEra Console & System
